# Welcome to HanStyle

HanStyle is a React component library providing various components that are highly customizable.
You can directly use the components with different functions and preset styles, which allows you to focus more on the
logic and functionalities of your application.

We built the component library with TypeScript, which makes it possible for you to enable TypeScript to perform
static type-checking.

Run `npm install hanstyle` in your console to install the component library.

Read the documentation of the components now by navigating to them in the sidebar.

You can also preview the components by clicking the 'Canvas' tab.
Further, you may control the props of each component using the 'Controls' tab, or the 'Control' column in the props table,
and get a preview of the component in real-time.
